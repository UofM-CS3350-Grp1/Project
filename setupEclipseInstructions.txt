installing windowbuilder:

1. go to: https://www.eclipse.org/windowbuilder/download.php

2. select your version of eclipse (I'm using kepler, if you're using another version there is a possibility that they are not compatible)

3. follow the instructions (I installed windowbuilder and SWT designer)

4. to open the window editor, right click on the java file you wish to edit, and open with > windowbuilder editor

5. If you want to learn more about using windowbuilder, read the documentation
